# Benutzersprache

Unter "Benutzersprache" kann der eingeloggte Benutzer eine gewünschte Benutzersprache festlegen.

| Option | Wert |
| - | - |
| Sprache durch Browser wählen | Ja oder Nein |
| Sprache | Englisch<br>Deutsch<br>Benutzerdefinierte Sprache |
